# Examples

This directory contains example scripts demonstrating how to use GMM-CT.

## Available Examples

### `basic_reconstruction.py`

Complete example showing:
- Geometry setup
- Parameter initialization
- Data generation
- Reconstruction
- Visualization

Run with:
```bash
python basic_reconstruction.py
```

## Creating Your Own Examples

1. Import the package:
```python
from gmm_ct import GMM_reco, ReconstructionConfig
```

2. Set up your geometry and configuration

3. Run reconstruction

4. Visualize results

See the provided examples for complete working code.
